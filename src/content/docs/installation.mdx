# Installation

This guide covers the complete installation process including dependencies, development setup, and editor configuration.

## System Requirements

| Requirement | Minimum | Recommended |
|------------|---------|-------------|
| Node.js | 18.0 | 20.x LTS |
| npm | 9.0 | 10.x |
| Disk Space | 200 MB | 500 MB |
| RAM | 2 GB | 4 GB |

## Step 1: Clone & Install

```bash
# Clone the repository
git clone https://github.com/your-org/velocity-pro.git my-saas
cd my-saas

# Install dependencies
npm install

# Start development server
npm run dev
```

## Step 2: Editor Setup

### VS Code Extensions (Recommended)

Install these extensions for the best experience:

```json
{
  "recommendations": [
    "astro-build.astro-vscode",
    "bradlc.vscode-tailwindcss",
    "esbenp.prettier-vscode"
  ]
}
```

### TypeScript Configuration

The project uses strict TypeScript. The `tsconfig.json` is pre-configured:

```json
{
  "extends": "astro/tsconfigs/strict",
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "react"
  }
}
```

## Step 3: Verify Installation

Run the build command to verify everything works:

```bash
npm run build
```

You should see output like:

```
 generating static routes
▶ src/pages/index.astro
▶ src/pages/features.astro
...
✓ Completed in 2.5s
```

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm run dev` | Start development server on port 4321 |
| `npm run build` | Build for production (static output) |
| `npm run preview` | Preview the production build locally |

## Troubleshooting

### Port already in use

```bash
# Use a different port
npm run dev -- --port 3000
```

### Node.js version issues

```bash
# Check your Node version
node --version

# Use nvm to switch versions
nvm use 20
```

> **Tip:** If you encounter any issues, check the [Astro docs](https://docs.astro.build) or open an issue on GitHub.
