# Components

Velocity Pro includes **48 production-ready components** organized into 13 categories. All components support dark mode, responsive design, and theme customization.

## Navigation (4)

### Navbar
The main navigation bar with logo, links, dropdown menus, theme toggle, and CTA button. Becomes a hamburger menu on mobile.

```astro
---
import Navbar from '../components/Navigation/Navbar.astro';
---
<Navbar />
```

### MobileMenu
Full-screen mobile navigation with smooth enter/exit animations.

### NavDropdown
Dropdown menu for nested navigation items with hover delay.

### Breadcrumbs
Contextual breadcrumb navigation for nested pages.

## Hero Sections (4)

### HeroHome
The primary hero with word-by-word text reveal animation, gradient orb background, email capture input, and social proof stats.

### HeroSimple
Clean, centered hero for interior pages with title, subtitle, and optional CTA.

### HeroSplit
Two-column hero with content on one side and image/illustration on the other.

### HeroWithVideo
Hero section with an embedded or modal video player.

### FeatureBento
Asymmetric bento-box layout for highlighting features with varying importance.

## Component Architecture

All components follow these patterns:

- **CSS Modules** for styling (zero Tailwind)
- **CSS Custom Properties** for theming
- **React islands** for interactive components
- **Astro components** for static layouts
- **TypeScript** throughout
